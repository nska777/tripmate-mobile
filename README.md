# TripMate AI Guide — мобильное приложение Android/iOS

Готовый MVP мобильного приложения для туристов, которые приехали в незнакомый город и не знают, куда идти.

## Что уже есть

- Expo / React Native / TypeScript
- Один код под Android и iOS
- Выбор города: Ташкент, Самарканд, Бухара, Хива
- Выбор времени поездки: 3 часа, 1 день, 2 дня, 3 дня
- Выбор бюджета: эконом, комфорт, премиум
- Выбор интересов: еда, история, фото, шопинг, семья, романтика, природа, вечер
- Генерация маршрута
- Расчет примерного бюджета
- Карточки мест с советами и безопасностью
- Фразы для туриста на местном языке
- Экран AI‑помощника с локальной логикой-заглушкой
- Сохраненные маршруты
- Нижнее мобильное меню

## Как запустить на компьютере

1. Установи Node.js LTS.
2. Открой папку проекта в VS Code.
3. В терминале выполни:

```bash
npm install
npm start
```

4. Установи приложение Expo Go на телефон.
5. Отсканируй QR-код из терминала.

## Запуск Android

```bash
npm run android
```

Можно открыть через Android Studio Emulator или через Expo Go на телефоне.

## Запуск iOS

```bash
npm run ios
```

На Windows iOS-симулятор не запустится, но можно открыть через Expo Go на iPhone или собрать через EAS Build.

## Сборка APK для Android

1. Установи EAS CLI:

```bash
npm install -g eas-cli
```

2. Авторизуйся:

```bash
eas login
```

3. Инициализируй проект:

```bash
eas init
```

4. Собери APK:

```bash
npm run build:android
```

## Сборка iOS

Для полноценной сборки iOS нужен Apple Developer Account.

```bash
npm run build:ios
```

## Где менять города и места

Файл:

```txt
src/data/cities.ts
```

Там можно добавлять города, маршруты, цены, советы, фразы, места.

## Где подключать реальный AI

Сейчас AI‑помощник работает как локальная заглушка в `App.tsx` внутри функции `sendChat`.

Позже туда можно подключить backend:

```ts
const response = await fetch('https://your-api.com/ai/travel-chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ message: text, city: route.city, route })
});
```
